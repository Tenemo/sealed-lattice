    <p>

    </p>

  <div class="landing-steps-shell">
    <Steps>
      1. **Install the public package name.**
         Use `sealed-lattice` as the only supported public import path.
      2. **Treat the runtime facade as intentionally empty.**
         The package exists today to stabilize packaging, docs, and release flow.
      3. **Develop inside the workspace packages.**
         Use the private packages for protocol, crypto, wasm, and testkit work.
      4. **Keep the Rust/WASM path verified.**
         The placeholder kernel must continue to build natively and load in Node and browsers.
    </Steps>
  </div>
