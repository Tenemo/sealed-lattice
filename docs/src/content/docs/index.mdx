<section class="landing-section not-content">
  <div class="landing-section__header">
    <h2>Follow the current milestone</h2>
    <p>
      The package is intentionally narrow. The supported path is short, explicit,
      and designed to keep the future architecture flexible.
    </p>
  </div>

  <div class="landing-steps-shell">
    <Steps>
      1. **Install the root package.**
         Use `sealed-lattice` as the only committed public entry point.
      2. **Hash strings or bytes.**
         Call `sha256Hex(...)` with UTF-8 text or `Uint8Array` input and expect
         lowercase hexadecimal output.
      3. **Handle missing runtime support explicitly.**
         If native digest support is unavailable, the package throws
         `UnsupportedRuntimeError` instead of silently falling back.
      4. **Treat future PQ capabilities as internal for now.**
         Serialization, transport, threshold flows, proofs, and wider lattice
         modules stay unpublished until their misuse-resistant boundaries are
         stable enough to freeze.
    </Steps>
  </div>
</section>
